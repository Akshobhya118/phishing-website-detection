#  Phishing Website Detection

A Machine Learning project that detects whether a website is **phishing** or **legitimate** using cybersecurity features.

##  Overview
Phishing websites try to steal sensitive user data. This system analyzes website attributes and classifies sites as safe or malicious.

##  Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Random Forest Algorithm  

##  Model Performance
Achieved **~96% accuracy** in phishing detection.

##  How to Run
```bash
pip install pandas numpy scikit-learn
python main.py
